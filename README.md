# AEON CRM Minimal

Stripped-down export of the AEON CRM shell for a secondary VPS.

Included routes:

- `/lists`
- `/dialer`
- `/users`
- `/users/statistics`
- `/leads`
- `/leads/search`
- `/campaigns`
- `/campaigns/stats`

Build steps:

1. `npm install`
2. `npm run build`
3. `npm run start`

Default landing route redirects to `/users`.
